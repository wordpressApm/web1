WEB

World Wide Web (WWW 또는 웹)은 문서 및 기타 웹 자원이 URL (Uniform Resource Locators)에 의해 식별되고 하이퍼 텍스트 링크로 연결되며 인터넷을 통해 액세스 할 수있는 정보 공간입니다. 영국 과학자 팀 버너스 - 리 (Tim Berners-Lee)는 1989 년에 월드 와이드 웹 (World Wide Web)을 발명했습니다. 그는 스위스의 CERN에서 근무하면서 1990 년에 최초의 웹 브라우저 컴퓨터 프로그램을 저술했습니다. [3] 웹 브라우저는 1991 년 1 월 시작한 다른 연구 기관과 1991 년 8 월 인터넷 일반인에게 1991 년 CERN 외부로 배포되었습니다.
