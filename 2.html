CSS

CSS (Cascading Style Sheets)는 마크 업 언어로 작성된 문서의 표현을 설명하는 데 사용되는 스타일 시트 언어입니다. HTML 및 XHTML로 작성된 웹 페이지 및 사용자 인터페이스의 시각적 스타일을 설정하는 데 가장 많이 사용되지만이 언어는 일반 XML, SVG 및 XUL을 비롯한 모든 XML 문서에 적용 할 수 있으며 음성 또는 기타 형식의 렌더링에 적용 할 수 있습니다 미디어. HTML 및 JavaScript와 함께 CSS는 대부분의 웹 사이트에서 시각적으로 매력적인 웹 페이지, 웹 응용 프로그램의 사용자 인터페이스 및 많은 모바일 응용 프로그램의 사용자 인터페이스를 만드는 데 사용되는 기본 기술입니다.
